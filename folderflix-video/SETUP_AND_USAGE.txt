FolderFlix - Setup and Usage

Required Software
-----------------
Install Node.js 18 or newer.

Check your installed Node.js version:
node -v

No extra npm packages are required. This project uses only built-in Node.js modules.

How To Run
----------
1. Open the project folder in CMD, PowerShell, Terminal, or VS Code Terminal.
2. Run:
   npm start

   Or double-click:
   start.bat

3. Open this URL in Chrome or Microsoft Edge:
   http://localhost:3000

4. Click Folder or Choose Folder.
5. Select the folder that contains your videos.

What It Does
------------
- Builds a playlist from supported video files in the selected folder.
- Ignores photos, HTML, CSS, JS, JSON, and other non-video files.
- Saves the last watched video and playback time.
- Restores the last playlist when browser permission is still available.
- Lets you delete the current video or a playlist item.
- Removes deleted videos from the playlist immediately.
- Moves to the next video automatically after deletion.
- Lets you remove the selected folder from the playlist without deleting files from disk.
- Saves watch/delete history in player-log.json.
- Clears player-log.json after folder removal to avoid old playlist state conflicts.

Supported Video Extensions
--------------------------
.mp4, .mkv, .webm, .mov, .m4v, .avi, .wmv, .flv, .ts, .m2ts, .3gp, .ogv

Important Warning
-----------------
The Delete feature really deletes the selected video file from your disk.
